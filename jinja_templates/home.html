Lose Little.

Win Big.

Welcome to Bit Share. This website allows users to each put in the same amount of bitcoin into a jackpot. Once that is completed the jackpot is then randomly distributed among each of the users who originally bet. Each user will get at least some bitcoin back. Some will win big. Some will win small. Some will break even. Some will lose a little. The point, is that you can win big, but only lose little. Enjoy!
